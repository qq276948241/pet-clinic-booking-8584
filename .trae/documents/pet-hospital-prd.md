## 1. 产品概述

宠物医院在线预约系统，为宠物主人提供便捷的线上预约挂号服务，涵盖科室浏览、医生选择、时段预约及预约管理全流程。
- 解决宠物主人电话预约不便、到院排队等候时间长的问题
- 目标用户：有宠物就医需求的宠物主人

## 2. 核心功能

### 2.1 用户角色

| 角色 | 注册方式 | 核心权限 |
|------|----------|----------|
| 宠物主人 | 无需注册，直接使用 | 浏览科室、预约挂号、管理预约 |

### 2.2 功能模块

1. **首页**：医院介绍、热门科室展示、快速预约入口
2. **预约挂号页**：科室选择、医生选择、时间段选择、提交预约
3. **我的预约页**：预约记录列表、取消预约

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 首页 | Hero 区域 | 展示医院名称、标语、背景装饰 |
| 首页 | 医院介绍 | 医院简介文字、特色服务标签 |
| 首页 | 热门科室 | 科室卡片列表（图标+名称+简介），点击可跳转预约 |
| 首页 | 快速预约入口 | CTA 按钮引导进入预约挂号页 |
| 预约挂号页 | 科室选择 | 横向/网格科室卡片，选中高亮 |
| 预约挂号页 | 医生选择 | 根据所选科室展示医生列表（头像+姓名+职称+简介） |
| 预约挂号页 | 时间段选择 | 日历选择日期 + 可用时间段网格 |
| 预约挂号页 | 提交预约 | 宠物信息填写（姓名+种类）、确认提交 |
| 我的预约页 | 预约记录列表 | 卡片式展示每条预约（科室、医生、时间、状态） |
| 我的预约页 | 取消预约 | 对待就诊的预约提供取消按钮，确认后取消 |

## 3. 核心流程

用户打开首页浏览医院信息和热门科室 → 点击预约挂号或科室卡片 → 进入预约挂号页选择科室 → 选择该科室下的医生 → 选择就诊日期和时间段 → 填写宠物信息 → 提交预约 → 在我的预约页查看预约记录 → 如需取消可点击取消按钮

## 4. 用户界面设计

### 4.1 设计风格

- 主色调：浅蓝 (#4DACFF / #E8F4FD) + 白色 (#FFFFFF)
- 辅助色：深蓝 (#1A73E8) 用于强调，浅灰 (#F5F7FA) 用于背景
- 按钮风格：圆角按钮，hover 时轻微阴影加深
- 字体：Noto Sans SC，标题 20-28px，正文 14-16px
- 布局风格：圆角卡片式布局，顶部固定导航栏
- 图标风格：lucide-react 线性图标

### 4.2 页面设计概述

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|---------|
| 首页 | Hero 区域 | 渐变浅蓝背景，大标题白色文字，装饰性宠物爪印图案 |
| 首页 | 医院介绍 | 白色圆角卡片，左侧图标+右侧文字 |
| 首页 | 热门科室 | 网格布局，圆角卡片悬停阴影动画 |
| 预约挂号页 | 步骤指示器 | 顶部横向步骤条（1-2-3-4），当前步骤高亮 |
| 预约挂号页 | 科室选择 | 2x3 网格卡片，选中边框变为主色调 |
| 预约挂号页 | 医生选择 | 水平滑动卡片列表 |
| 预约挂号页 | 时间段选择 | 日期横向滚动 + 时间段网格 |
| 预约挂号页 | 提交区域 | 底部固定提交按钮 |
| 我的预约页 | 预约卡片 | 圆角卡片，左侧状态色条，右侧操作按钮 |
| 我的预约页 | 取消确认 | 模态弹窗确认取消操作 |

### 4.3 响应式设计

- 桌面优先设计，最大宽度 1200px 居中
- 移动端适配：科室网格 2 列变 1 列，卡片间距缩小
- 触摸优化：按钮最小 44px 点击区域
